# movies

Aplicación web para explorar un catálogo de películas y gestionar favoritas.

## Funcionalidades
- Listado de películas
- Marcar y desmarcar películas como favoritas
- Filtro por nombre
- Filtro por año de lanzamiento
- Persistencia de favoritos con LocalStorage

## Tecnologías
- HTML
- CSS
- JavaScript (ES6+)
- LocalStorage

## Aprendizajes
- Manipulació del DOM
- Manejo de estado en frontend
- Estado de la aplicación
- Uso de LocalStorage
- Arquitectura básica en JavaScript
- Filtros y renderizado dinámicorquitectura básica en JavaScript
- Organización básica del código
